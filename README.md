# React + TypeScript + Vite

This is a React application built with TypeScript and Vite.  
The project uses:

- **Tailwind** for UI components and styling (**https://tailwindcss.com**)

---

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 1. Start the development server

```bash
npm run dev
```

The app will run at **http://localhost:5173** (or whichever port is shown in the terminal).
